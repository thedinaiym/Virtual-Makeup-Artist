## 📌 О проекте (Abstract)
**Проект - Итоговый проект по Deep Learning**

Большинство существующих моделей нейросетевого макияжа (например, BeautyGAN) осуществляют «слепой» перенос стиля: они копируют макияж с референсного изображения на целевое, не учитывая анатомию лица, цветотип кожи и правила визажа. 

Данный проект предлагает **новую эстетически-осведомленную (aesthetics-aware) архитектуру**, написанную с нуля. Система объединяет классическое компьютерное зрение (Rule-Based Expert System) для создания идеальной обучающей выборки и кастомную генеративную состязательную сеть (cGAN) для быстрого и легковесного переноса макияжа с учетом индивидуальных черт лица пользователя.

## 🧠 Научная новизна (Research Novelty)
Проект претендует на научную новизну в двух аспектах:
1. **Синтетическая генерация данных на основе правил экспертов (Rule-Based Synthetic Data):** Решение проблемы отсутствия парных датасетов ($X, Y$) путем программной генерации макияжа. Алгоритм вычисляет форму лица и цветотип кожи для подбора идеального контуринга и оттенков, а затем накладывает их с помощью *Poisson Image Editing* для реалистичного смешивания.
2. **Архитектурная модификация генератора (Mask-Conditioned U-Net with AdaIN):** * Отказ от классического 3-канального (RGB) входа. Модель принимает на вход многоканальный тензор, включающий бинарные маски лица (губы, глаза, контур), что дает сети жесткие пространственные подсказки и избавляет от артефактов на фоне (волосы, одежда).
   * Интеграция слоев *Adaptive Instance Normalization (AdaIN)* для плавного контроля интенсивности макияжа (от легкого нюда до вечернего).

## 🗄 Исходные данные (Datasets)
В качестве базовых лиц (Source Images - $X$) будут использоваться открытые высококачественные датасеты:
* **FFHQ (Flickr-Faces-HQ):** Основной выбор. Содержит 70,000 изображений лиц высокого качества в разрешении до 1024x1024. Имеет огромное разнообразие возрастов, этносов и условий освещения.
* **CelebA-HQ:** Альтернативный датасет на 30,000 изображений. 

## 🏗 Архитектура системы

### 1. Пайплайн данных (Data Generation Pipeline)
* **Face Parsing:** Использование `MediaPipe Face Mesh` (468 3D-точек) для сегментации зон лица на базовых фото из FFHQ.
* **Expert Logic (Колористика и Геометрия):** * Анализ формы лица $\rightarrow$ выбор карты контуринга.
  * Анализ RGB-значений кожи $\rightarrow$ определение подтона $\rightarrow$ подбор палитры макияжа.
* **Рендеринг (OpenCV):** Наложение цветовых полигонов с использованием размытия по Гауссу и `cv2.seamlessClone`.
* **Выход:** Идеально выровненные пары $(X, Y)$ для обучения.

### 2. Нейросетевая архитектура (Deep Learning Architecture)
* **Генератор (Custom U-Net):**
  * **Input:** $N \times C \times H \times W$, где $C = 5$ (RGB + Маска губ + Маска кожи).
  * **Bottleneck:** Внедрение AdaIN для управления вектором стиля.
  * **Skip-Connections:** Сохранение высокочастотных деталей лица.
* **Дискриминатор (PatchGAN):** Анализирует локальные патчи изображения (например, $70 \times 70$ пикселей) для детализации текстур.

### 3. Функция потерь (Loss Functions)
Обучение строится на композитной функции потерь:
$$\mathcal{L}_{total} = \lambda_1 \mathcal{L}_{cGAN} + \lambda_2 \mathcal{L}_{L1} + \lambda_3 \mathcal{L}_{Perceptual}$$

## 💻 Аппаратное обеспечение и стек (Hardware & Tech Stack)
* **ОС:** Ubuntu 22.04.5 LTS (Native Linux environment)
* **CPU & RAM:** Intel Core i9-14900K, 32 GB RAM. Обеспечивает сверхбыстрый препроцессинг данных и генерацию синтетического датасета.
* **GPU:** **NVIDIA GeForce RTX 4080 SUPER (16GB VRAM)**. Флагманская вычислительная мощность тензорных ядер (Ada Lovelace) гарантирует быстрое обучение GAN. 16 ГБ видеопамяти позволяют использовать оптимальные размеры батчей (Batch Size: 8-16) при высоком разрешении изображений (256x256 или 512x512).
* **Язык:** Python 3.10+
* **Фреймворк:** PyTorch (с поддержкой CUDA 13.1), Torchvision
* **Computer Vision:** OpenCV, MediaPipe
* **Трекинг экспериментов:** Weights & Biases (W&B)
